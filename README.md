# ABSA Mobile Shop

ระบบวิเคราะห์ความคิดเห็นลูกค้าด้วยเทคนิค **Aspect-Based Sentiment Analysis (ABSA)
สำหรับร้านขายเคสมือถือ โดยใช้โมเดลภาษาไทย


## Features

- วิเคราะห์ sentiment จากข้อความภาษาไทย
- แยกผลลัพธ์ตามแต่ละ aspect:
  - สินค้า (Product)
  - ราคา (Price)
  - บริการ (Service)
  - ความหลากหลาย (Variety)
- แสดงผลเป็นคะแนน / label (Positive, Neutral, Negative)
- มีระบบ evaluation และ report
- มี Web App ด้วย Streamlit
